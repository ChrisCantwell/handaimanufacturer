# Contributing

Thank you for your interest in HandAIManufacturer.

This project is in early-stage development. Issues, testing notes, feature requests, printer/camera compatibility reports, and design discussion are welcome.

## Current contribution status

Pull requests may not be accepted until a formal contribution policy is finalized.

The project may later add a Contributor License Agreement (CLA), Developer Certificate of Origin (DCO), or another contribution process that preserves the maintainer's ability to choose future licensing and distribution options.

## Why this policy exists

HandAIManufacturer may eventually have both open-source and commercial use cases. While the project is young, the maintainer wants to avoid accidentally accepting outside code in a way that complicates future licensing, packaging, or commercial distribution.

This is not meant to discourage discussion. It is meant to keep the legal and project structure clean while the architecture is still forming.

## Welcome contributions for now

Good early contributions include:

- Bug reports.
- Printer compatibility notes.
- Moonraker behavior notes.
- OrcaSlicer CLI findings.
- RTSP/ONVIF camera compatibility notes.
- Documentation suggestions.
- Safety concerns.
- Feature requests.
- Real-world print workflow reports.

## Please do not submit yet

Until the contribution policy is finalized, please avoid submitting large code pull requests or third-party code drops.

## Security and safety

Do not post secrets, private IPs, camera credentials, API keys, OAuth tokens, or other sensitive configuration in issues or pull requests.

This software may control equipment that heats up and moves. Safety concerns are welcome and should be treated seriously.
